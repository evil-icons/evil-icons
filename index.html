Evil Icons

Evil Icons is an SVG icons set designed for using in web projects. Rails gem, npm package and bower package are available.

Sponsored by Evil Martians.

Usage

Rails

Add 'evil_icons' gem to your Gemfile:

  gem 'evil_icons'

Add the evil icons styles to your application.css:

  /*= require evil-icons */

Render the evil icons sprite in your templates:

  <%= evil_icons %>

After these steps you can render icons using theicon helper. Here are some examples: 

<%= icon 'ei-search' %>
<%= icon 'ei-arrow-right', size: :m %>
<%= icon 'ei-envelope', size: :l %>

An icon's color can be changed in CSS: 

.icon {
  fill: green;
}
.icon--ei-facebook {
  fill: blue;
}
